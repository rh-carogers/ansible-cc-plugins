# ansible-agile-coach

Agile product ownership coaching and user story expertise for Ansible teams.

## Skills

- **ansible-product-owner** - Coaching on user stories, story splitting, backlog refinement, INVEST criteria, acceptance criteria, and agile product development practices.

## Installation

```bash
claude plugins marketplace add rh-carogers/ansible-cc-plugins
claude plugins install ansible-agile-coach@ansible-cc-plugins --scope user
```
