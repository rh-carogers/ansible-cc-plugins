# ansible-agile-coach

Agile coaching, Jira expertise, and product ownership guidance for Ansible teams.

## Skills

- **ansible-product-owner** - Coaching on user stories, story splitting, backlog refinement, INVEST criteria, acceptance criteria, story mapping, anti-patterns, and agile product development practices. Includes reference materials from the Scrum Guide, Mike Cohn's *User Stories Applied*, Roman Pichler's *Agile Product Management with Scrum*, and Jeff Patton's story mapping approach.
- **ansible-jira-helper** - Expert guidance on Ansible-specific Jira workflows and best practices. Covers issue types and hierarchy (Strategic Goals through Sub-tasks), backlog prioritization, feature delivery processes, support escalation workflows, and unified backlog ranking methodology. Includes reference summaries from Ansible's internal Jira guides.

## Installation

```bash
claude plugins marketplace add rh-carogers/ansible-cc-plugins
claude plugins install ansible-agile-coach@ansible-cc-plugins --scope user
```
